# React Native weather app

## Tech requirements

- React Native
- Material UI
- TypeScript
- Mobx state manager
- i18n multilang support
- External API (e.g. [Weatherapi](https://www.weatherapi.com/>))
- Release build pipeline

## Features

- Several screens
- Search city by name
- City favorites list
- City weather details
- City 7-day forecast
- Offline-mode
